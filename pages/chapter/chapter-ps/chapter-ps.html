站点管理与页面属性

掌握Dreamweaver中站点创建与管理的基本操作

本地站点创建

创建步骤

  1. 运行Dreamweaver，选择菜单"站点→新建站点"命令，弹出"站点设置对象"对话框
  2. 选择"站点"选项，在"站点名称"文本框中输入站点名称
  3. 单击"本地站点文件夹"后面的"浏览文件夹"按钮，在打开的"选择根文件夹"对话框中选择文件夹
  4. 选择"高级设置"中的"本地信息选项"，单击"默认图像文件夹"后的"浏览"按钮，选择图像文件夹
  5. 在"站点设置对象"对话框中单击"保存"按钮
注意：站点名称应使用英文或数字，避免使用中文和特殊字符，以确保兼容性。

站点导出与导入

导出站点

  1. 选择菜单"站点→管理站点"命令，打开"管理站点"对话框
  2. 选择要导出的站点名称，单击"导出站点"按钮
  3. 在弹出的"导出站点"对话框中，选择保存位置并输入文件名
  4. 单击"保存"按钮完成导出

导入站点

  1. 选择菜单"站点→管理站点"命令，打开"管理站点"对话框
  2. 单击"导入站点"按钮，弹出"导入站点"对话框
  3. 选择站点定义文件(.ste文件)
  4. 单击"打开"，然后在"管理站点"对话框中单击"完成"按钮

页面属性设置

页面属性包括外观、链接、标题、标题/编码、跟踪图像等分类设置。

设置步骤

  1. 在属性面板单击"页面属性"按钮，打开"页面属性"对话框
  2. 在"分类"列表中选择要设置的类别
  3. 设置相关参数：
    • 链接颜色：设置超链接默认颜色
    • 已访问链接：设置访问过的链接颜色
    • 变换图像链接：设置鼠标悬停时的链接颜色
    • 活动链接：设置活动状态的链接颜色
  4. 将"下划线样式"设置为需要的样式
  5. 单击"确定"按钮应用设置
